# Note-Pad

 ## This Note Pad App can do 
- Open File from pc
- Save File to pc
- Print the file
- Copy text
- Cut text
- Paste text
- Change the font
- Bold
- Italic
- Underline
- Left align
- Center align
- Right align
- Zoom in
- Zoom out
- Change back ground color
- Change the text color
